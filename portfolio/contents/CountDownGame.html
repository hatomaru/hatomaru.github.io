description 概要

フィールドに散らばる図形に触れてお題の数字を0よりも少なくするゲームです。
違う形に触れ続けるとコンボ！獲得できるスコアがアップします。しかし、同じ形に連続して触れてしまうとコンボが途切れてしまいます。

ひとりでもみんなとでも遊べる全3ステージで、ハイスコアを目指していくミニゲームです。

build 使用した技術

Communities
Clusterでのゲーム開発
メタバースにゲームを配信するための基礎を制作を通じて学べました。
build
Blender
ゲームに登場する図形をBlenderでモデリングしました。

psychology_alt なぜ作ったか（Why）

みんなが集まって遊べるアスレチックのようなミニゲームがあれば、
メタバースの世界に触れてみたいと感じる人がいるのではと思ったから。

lightbulb どう解決したか

多くの人々に遊んでもらうために「多言語対応が必要」という課題にぶつかりました。
この課題はゲーム内のUIを日本語と英語両方に対応することで解決しました。

school 学び

メタバースでのゲーム開発は普段と異なる感覚で多くの学びを経験することが出来ました
一方、ゲームの見せ方の部分が不足していたので、分かりやすく奥行きができるようにステージ構成やギミックを調整したいと思いました。
